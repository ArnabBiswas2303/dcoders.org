COMPETITION

Mission

Your Task

Your task, as a team of three is to design a new look for the Suchana Webpage shown below. You need to stylize it and turn it around into an awesome looking single page website! The team having the best-looking website, as selected by the Judges, WINS!

Criteria

Your Website should meet the following criteria and have the following sections for a better chance of winning:
  1. Notification Center:-An area to display all recent notifications or Suchanas
  2. Navigation Bar:-A Navigation bar containing the links <> and a dummy search bar
  3. Suchana Login Section:- A link to a seperate Login Page via a button or a seprate section on the Home Page itself to Login
  4. Responsive [Optional] :- Website should resize with browser window size and with device. Use Bootstrap to do this easily.
  5. A Gallery :- A gallery containing pictures of SMIT. You can use a carousel for this if you want
  6. An Intuitive Way to Display Teacher Information:- Teacher Information such as photo, name, contact number and email should be accessible easily. Take 3 teachers as a test case.

Judging

The Top 4 Websites will be selected for a Short Coding Round. Top 4 teams will be selected by the following criteria:
  1. Asthetic Appearence:-How it will look
  2. Intuitivity :-How easy is the site to operate. Responsive Sites get Bonus Points.
  3. CSS Techniques Used:-How you are designing the CSS
  4. Performance in the Question & Answer Round :- Judges might ask you questions regarding your website like how it has been designed.
